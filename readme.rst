PTX-RPC |build-status| |codacy|
===============================

PTX-RPC is an asymmetric Python RPC Library with a REST-like API and JSON-RPC transport.

|license|

Installing
==========

Console commands::

   python setup.py install

Requirements
============

* Python 2.7
* json
* requests
